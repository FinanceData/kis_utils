# KIS Utils Agent Skill

한국투자증권(KIS) API를 활용하여 주식 시세 조회, 계좌 잔고 확인 및 주식 주문을 수행하는 스킬입니다.

## 주요 도구 (Tools)

### 1. 시세 조회
- **`price(stock_code)`**: 특정 종목의 현재가, 전일대비, 거래량 등을 조회합니다.
- **`daily(stock_code, ...)`**: 특정 종목의 일/주/월별 시세 데이터를 조회합니다.

### 2. 계좌 관리
- **`balance()`**: 현재 계좌의 예수금 정보와 보유 종목 리스트를 가져옵니다.
- **`order_history()`**: 최근 주문 및 체결 내역을 확인합니다.
- **`buyable(stock_code, price)`**: 특정 가격으로 매수 가능한 최대 수량을 계산합니다.

### 3. 거래 수행
- **`buy(stock_code, quantity, price, ...)`**: 주식 매수 주문을 넣습니다.
- **`sell(stock_code, quantity, price, ...)`**: 주식 매도 주문을 넣습니다.

### 4. 기타
- **`token()`**: API 접근 토큰을 발급하거나 갱신합니다.

## 에이전트 가이드 (Guidelines)
- 사용자가 종목명을 말하면 먼저 `price` 도구를 사용하여 정확한 종목코드를 확인하거나 시세를 조회하세요.
- 주문을 수행하기 전에는 반드시 `buyable` 또는 `balance`를 통해 수량이 적절한지 확인하는 것이 좋습니다.
- 모든 금액과 수량 데이터는 숫자로 처리되므로 계산이 용이합니다.
- 보안이 필요한 작업(주문 등) 시에는 사용자에게 최종 확인을 받는 과정을 거치는 것을 권장합니다.
